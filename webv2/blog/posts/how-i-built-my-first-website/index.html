How I Built My First Personal Website

Arbin Niroula · 5 min read · 2026

Building my first personal website was one of the most exciting and rewarding things I've done as a developer. It forced me to bring together everything I'd been learning — HTML structure, CSS styling, and JavaScript interactivity — into one coherent project that represented me.

Starting with HTML

I started by learning the basics of HTML, which allowed me to structure content using headings, paragraphs, links, and images. The key insight early on was that HTML is just the skeleton — it tells the browser what things are, not how they should look.

I structured my site with a clear hierarchy: a navigation bar at the top, a hero section to introduce myself, sections for my skills and projects, and a contact area at the bottom.

Making It Look Good with CSS

Once I had a clear structure, CSS transformed everything. I learned about the box model, flexbox layouts, CSS variables for consistent colors, and media queries for mobile responsiveness. Getting the site to look identical on both a 1440px desktop and a 375px phone was genuinely satisfying.

I chose a color scheme that felt personal — dark backgrounds with warm accent colors — and picked fonts from Google Fonts that felt editorial rather than generic.

Adding Life with JavaScript

JavaScript came next. I used it to create smooth scrolling between sections, a mobile hamburger menu that toggles open and closed, and small entrance animations triggered when elements scroll into view using the Intersection Observer API. These small touches made the site feel alive.

Choosing a Background and Layout

One of the most fun parts was personalising the visual design. I chose a background that matched my aesthetic, arranged sections to flow naturally, and made sure each section had breathing room.

What I Learned

Through this process, I not only improved my coding skills but also understood how design, usability, and functionality come together in web development. It was a small project, but it gave me the confidence to build more complex sites and helped me develop a genuine aesthetic sense.

If you're thinking about building your own personal site — just start. It doesn't have to be perfect. It just has to be yours.
