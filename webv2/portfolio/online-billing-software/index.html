About This Project

A fully functional online billing software built with vanilla HTML, CSS, and JavaScript. Designed to help small businesses create, manage, and print invoices without any backend dependency — everything runs in the browser.

The app supports adding line items, auto-calculating totals and taxes, customer information entry, and generating a clean printable invoice layout. It was built with simplicity and practicality in mind.

Key Features

  • Add/remove line items dynamically
  • Automatic subtotal, tax, and total calculation
  • Customer and business info entry
  • Print-optimised invoice layout
  • No backend required — fully client-side
  • Mobile-responsive design

What I Learned

This project deepened my understanding of DOM manipulation, dynamic content rendering, and print stylesheets in CSS. It also taught me how to think about user workflow — what a real business user actually needs when generating invoices.
